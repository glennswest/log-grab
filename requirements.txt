# GUI Requirements:
# tkinter is required for the GUI and should include Tcl/Tk 8.6+ (preferably 9.0.2+)
#
# Quick Setup:
# Run: ./setup_tkinter.sh
#
# Manual Installation:
# - macOS: brew install tcl-tk && brew install python-tk
# - Ubuntu/Debian: sudo apt-get install python3-tk tk-dev tcl-dev
# - Windows: tkinter usually included with Python from python.org
#
# For Tcl/Tk 9.0.2 specifically:
# - macOS: Use Homebrew (automatically gets latest version)
# - Linux: May need to compile from source or use newer distro
# - Windows: Download Python 3.11+ from python.org
